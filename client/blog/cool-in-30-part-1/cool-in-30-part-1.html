How to Make S*** Look Cool in 30 seconds

This is a series aimed at developers who have little knowledge of design principles, but want to add a little extra pizazz to their sites. The tips given are based off of current web design trends and should make anyone who looks at your site go "Holy F*** you've got style"

In this first entry in the "How to Make S*** Look Cool in 30 seconds" series, we are going to talk about full-bleed, blurred background images. This technique is so hot in web design right now, and is a super simple way to make your project look snappy.

How to do it:

You'll need a few things: an html file, a css file, and an image. That's it! The image can be whatever you want, it can be something that relates to the project you're working on, or you can just pull a random image to use. Here are some nice royalty free sites you can take images from:

  • - Unsplash
  • - Gratisography
  • - Splitshire

The HTML

<body>
   <h1>This S*** Looks Hot</h1>
   <div class="blurred-background"></div>
</body>

The CSS

h1 {
   font-family: 'Courier New', sans-serif;
   text-align: center;
   margin-top: 50px;
   font-size: 60px;
}
.blurred-background {
   background: url('bridge.jpg')
      no-repeat center center fixed;
   background-size: 110%;
   -webkit-filter: blur(5px);
   filter: blur(5px);
   width: 100%;
   height: 100%;
   position: absolute;
   top: 0;
   left: 0;
   z-index: -1;
}
