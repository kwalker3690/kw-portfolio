Github Scout allows you to view Github data for users or languages at a glance. Along with three other developers, we built the site using Node, MySql, Angular, and Sass. I played the role of designer as well as frontend developer, implementing my designs in Angular and Sass.

Github Scout allows you to view graphs about individual user's activity on Github over time. We pull this data directly from the Github API upon performing a search. You can also compare two users against each other, and see the graphs automatically update. The graphs are built with NVD3, a framework that integrates Angular and D3 beautifully.

Individual languages can be graphed on Github Scout. These graphs are made by using Google Big Query to pull data from Github Archive. You can also compare multiple languages against each other and again see the graphs automatically update.
